# The framework of big data technology
大数据架构详解-note
## 综述
例如：运营商数字化转型衍生业务
* SQM（运维质量管理）  
* CSE (客户体验提升)
* MSS（市场运维支撑）
* DMP（数据管理平台）
## 数据获取
info:数据获取涉及技术
* 探针（电信特有）  
  主要涉及到了IB`（infinite band）`技术
* 爬虫  
  最基本的系统的结构：  
  > 数据中心
  >> 服务器
  >>> 爬虫程序   
在数据中心的不同服务器里，协同工作的方式一般有：
  * 主从式
* Flume（采集日志数据）
* Kafka（消息中间件）
